# Marlyn Grullon – Product Engineer & UI/UX Designer Portfolio

## Overview
This repository contains the source code for my personal web portfolio. It serves as a digital showcase of my work at the intersection of design systems and intelligent interfaces. As a Computer Science undergraduate specializing in Artificial Intelligence and Human-Computer Interaction (HCI), the portfolio highlights my ability to own features end-to-end—taking products from initial Figma prototypes to production-ready components.

## Key Highlights
* **Core Focus:** UI/UX Design, Privacy-First AI, Edge Computing, and Full-Stack Development.
* **Featured Projects:**
* * **ResilientCare.ai:** A privacy-first Generative AI application built to run locally in the browser using WebAssembly and Transformers.js.
  * **Freedom Finance:** A secure, client-side personal finance tracker featuring interactive data visualizations and a local AI assistant (LangChain.js).
* **Technical Stack:** Built from the ground up using semantic HTML5, modern CSS (featuring token-based variables, responsive CSS Grid/Flexbox, and glassmorphism UI), and vanilla JavaScript (Intersection Observer for scroll-reveal and custom typing animations).
